## 🎓 Pro Tips

### ✅ **Daily Routine (15 minutes)**
```
Morning (5 min):   Review due items (flashcards)
Commute (10 min):  Driving Mode (hands-free learning)
Evening (5 min):   Learn 3-5 new words
```

### ✅ **Spaced Repetition Power**
```
Day 0:  Learn word → Mark as "Learning"
Day 1:  Review → Easy ✅ → Next review in 3 days
Day 4:  Review → Good ✅ → Next review in 7 days
Day 11: Review → Easy ✅ → Next review in 15 days
Day 26: Review → Easy ✅ → MASTERED! ⭐
```

### ✅ **Driving Mode Best Practices**
```
1. Setup before starting car
2. Connect to Bluetooth/AUX
3. Choose "Recently Learned" mode
4. Set speed to 0.9x (normal)
5. Press Play → Focus on road! 👀🚗
6. Use Pause for phone calls/traffic
```

## 🆘 Common Questions

**Q: How many words should I learn per day?**  
A: Start with 5-10 words. Focus on quality over quantity. Review is more important than adding new words.

**Q: When should I use Driving Mode?**  
A: During commute, traffic, or any time you can listen hands-free for 10-20 minutes.

**Q: How long to see results?**  
A: Week 1: Familiar with basics. Month 1: 50+ words. Month 3: Conversational foundation.

**Q: What if I miss a day?**  
A: No problem! The spaced repetition algorithm adjusts. Just review when you can.
